Pull request overview
This PR migrates the app off the Supabase SDK to a direct PostgreSQL-backed architecture using TanStack Start server functions, adds a custom JWT + bcrypt auth system, and updates deployment configs/docs for Vercel/Railway/Render.

Changes:

Added PostgreSQL schema + migration tooling and a shared pg pool helper for server functions.
Replaced Supabase data access with server functions (data-fns, admin-fns) and updated routes/admin UI to use them.
Implemented custom auth (JWT in localStorage, attach middleware, admin enforcement) and refreshed deployment documentation/configs.
